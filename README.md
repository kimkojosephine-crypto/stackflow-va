# StackFlow VA — The Purpose Metrics Tool-Discovery Engine

A Ruby on Rails web application that helps Virtual Assistants find their personalised Day-1 tool stack based on their niche and budget.

## Built With
- Ruby 3.2.2
- Rails 8.1.2
- SQLite3
- WSL2 (Ubuntu 24.04)

## Getting Started

### Prerequisites
- Ruby 3.2.2
- Rails 8.1.2
- SQLite3

### Installation

1. Clone the repo
   git clone https://github.com/kimkojosephine-crypto/stackflow-va.git
   cd stackflow-va

2. Install dependencies
   bundle install

3. Set up the database
   rails db:migrate

4. Start the server
   rails server

5. Visit http://localhost:3000/va_profiles/new

## How It Works
1. Select your VA niche (General, Creative, Social Media, Lead Gen, Real Estate, Tech, or Travel)
2. Enter your monthly tool budget
3. Get a personalised tool stack recommendation
4. Your profile is saved to the database for future reference

## VA Niches Supported
- General VA — Administrative & Executive Support
- Travel & Itinerary VA — Trip Planning & Bookings
- Creative VA — Content, Design & Brand Support
- Social Media VA — Digital Presence & Community Management
- Lead Gen VA — Sales Pipeline & Business Development
- Real Estate VA — Property Operations & Transactions
- Tech VA — AI, Automation & Systems Specialist
